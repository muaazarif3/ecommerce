# Responsive E-Commerce Platform

This project is a fully responsive e-commerce platform built using **React**, **Tailwind CSS**, **Redux Toolkit**, **Formik**, **Yup**, and **Makeup API**. It includes a landing page, product listings, product detail pages, a shopping cart, a multi-step checkout process, an order history page, and an admin panel with CRUD functionality for managing products and user accounts.

## Features
- **Landing Page**: Includes a hero section and featured products.
- **Product Listings**: Displays products with filtering and sorting options.
- **Detailed Product Pages**: Shows product details with images, pricing, and descriptions.
- **Shopping Cart**: Add/remove products with price calculation.
- **Multi-step Checkout**: Formik & Yup-powered form validation for shipping and payment information.
- **Order History**: Users can view their past orders.
- **API Integration**: Uses Makeup API to fetch data.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript, React.js, Tailwind CSS
- **State Management**: Redux Toolkit
- **Form Handling & Validation**: Formik and Yup
- **API**: Makeup API
- **Build Tool**: Vite

## Features Breakdown

### 1. **Landing Page**
   - Hero section with a call-to-action.
   - Display of featured products.

### 2. **Product Listings**
   - Grid layout with products fetched from an API.
   - Users can filter and sort products.

### 3. **Product Detail Page**
   - Shows detailed information about a product.
   - Users can add the product to the cart.

### 4. **Shopping Cart**
   - Users can view added products, adjust quantities, and remove items.
   - Total price calculation.

### 5. **Multi-step Checkout**
   - Step-by-step checkout process.
   - Formik used for form handling, Yup for validation.

### 7. **Order History**
   - Displays users' previous orders.

## API Integration

- The project integrates with **Makup Api** for fetching product data.
